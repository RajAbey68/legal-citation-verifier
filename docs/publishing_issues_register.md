## How to use this register

1. **Raise an issue** — add a row with the next ID, severity, owner, and date raised
2. **Update status** — edit the Status column as work progresses  
3. **Close** — move row to Closed Issues table with resolution summary and close date
4. **Link to risk** — if an issue reveals a gap in the risk register, add a new risk row in `publishing_risk_register.md`
